> [Leetcode 27](https://leetcode.com/problems/remove-element/description/)
> # 注意⚠️数组的元素在内存地址中是连续的，不能单独删除数组中的某个元素，只能覆盖。
> ```.py
> class Solution(object):
    def removeElement(self, nums, val):
        """
        :type nums: List[int]
        :type val: int
        :rtype: int
        """
        fast = 0
        slow = 0
        size = len(nums)

        while fast < size:
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        return slow

> ```
